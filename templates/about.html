About College-Space



College-Space is an online portal for college students. Students and faculties can upload Notes, Images or Post updates happening in college.

College-Space is built in Python. It uses Python's Flask library for backend server.

The frontend is built using HTML, CSS and JavaScript.

And the databases used are MySQL for raw data and Firebase for storing files like PDFs and Images.
